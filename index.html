Opserv provides you with an intuitive, elegant user interface that allows both
visual and textual presentation of gathered system metrics in real-time.

main features

cross-plattform

The OpServ monitoring project was build to run across different plattforms. Build with Python 3 we currently support Windows, Mac OS X and Linux. We will expose all the metrics your system offers.

easy to use

An easy-to-use installer and executable for every platform is planned. If you want to get the latest release, simply head over to our downloads page. No other databases, libraries or software required.

historical and live data

The project gathers different metrics at changeable rates. In addition to retrieving live data through one of the APIs you've full access to all measured data since you started the program, as data is persistently saved to an embedded database.

open source

The project and it's source code is published under the GPL 3.0 license. Feel free to share, use or modify the code as you desire. In case you're missing a feature make a pull request or create an issue on GitHub.
