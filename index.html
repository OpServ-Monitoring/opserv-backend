main features

cross-plattform


The OpServ monitoring project was build to run across different plattforms. Build with python 3 we currently support Windows, Mac OS X, Linux and Free BSD. Note however that some metrics may not be available on every plattform.

easy to use


Despite downloading and running the source code yourself you can download one of our pre-build packages, optimized for your plattform which ensures an easy use. See the downloads page for further instructions.

historical and
live data

The project gathers different metrics at changeable rates. In addition to retrieving live data through one of the apis you've full access to all measured data since the program launch as data is persisted to the embedded database.

open source


The project and it's source code is published under the GPL 3.0 license. Feel free to share, use or modify the code as you desire. In case you're missing a feature make a pull request or open a new issue on GitHub.
